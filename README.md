NovaCart-Supply-Chain-Analytics
End-to-end Supply Chain Analytics project using SQL, MySQL, Power BI and DAX with executive dashboards and business insights.

Project Overview

NovaCart Supply Chain Analytics is an end-to-end Business Analytics project that demonstrates how data can be transformed into actionable business insights using SQL, MySQL, Power BI, and DAX.

The project analyzes customer demand, product performance, warehouse operations, logistics efficiency, regional profitability, and executive KPIs through an interactive Power BI dashboard.

The dataset is inspired by real-world retail supply chain operations and was created for portfolio-based business analytics.

Business Problem

Supply chain organizations generate large volumes of operational data every day. Without effective analysis, it becomes difficult to identify:

- High-value customers
- Best-performing products
- Inventory priorities
- Warehouse efficiency
- Delivery performance
- Regional profitability
- Return patterns
- Executive KPIs for decision-making

This project demonstrates how business intelligence can transform raw operational data into meaningful business insights.

Dataset

The project consists of six relational tables:

Customers 
Orders
Products
Warehouses
Logistics
Returns

Data Model

The database was designed in MySQL Workbench using an Entity Relationship (EER) Diagram.

Relationships:

Customers → Orders
Products → Orders
Warehouses → Orders
Orders → Logistics
Orders → Returns
